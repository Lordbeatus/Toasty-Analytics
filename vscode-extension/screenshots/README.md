# Screenshots Folder

Add screenshots here for the VS Code Marketplace and README:

- `welcome.png` - Welcome screen screenshot
- `results.png` - Code quality results panel
- `status-bar.png` - Status bar indicator showing server connection
- `comprehensive.png` - Comprehensive analysis results

Recommended size: 1280x720 or similar HD resolution

## How to Take Screenshots

1. Press F5 to launch Extension Development Host
2. Trigger the various features (welcome screen, grading, etc.)
3. Use Windows Snipping Tool or similar to capture screenshots
4. Save them in this folder with the names above
5. Update the README.md image paths if needed

**Note:** Screenshots are optional but highly recommended for marketplace visibility!
